swapBlock(self, block, subBlockALocation, subBlockBLocation)

source code 

Function to swap 2 sub-blocks within a block. For example, if the block consist of [1, 'A', 2, 'B', 3]; swapping 'A' and 'B' will result in [1, 'B', 2, 'A', 3].

>>> import jigsaw
>>> c = jigsaw.JigsawCore()
>>> block = 'ABCDEFGHIJKLM'
>>> swapped = c.swapBlock(block, (3, 4), (10, 11))
>>> swapped
'ABCKEFGHIJDLM'
>>> reverse_swapped = c.swapBlock(swapped, (3, 4), (10, 11))
>>> reverse_swapped
'ABCDEFGHIJKLM'
>>>
Parameters:
  • block (string) - block to be processed.
  • subBlockALocation (tuple) - coordinate of first sub-block.
  • subBlockBLocation - coordinate of second sub-block.

shuffleBlock(self, block, length)

source code 

Function to take a block, divide the block into smaller sub-blocks, and shuffle the order of the sub-blocks.

>>> import jigsaw
>>> c = jigsaw.JigsawCore()
>>> text = 'MyNameIsMauriceLing,AndIInventedJigsawEncryption'
>>> (shuffled_blocks, block_order) = c.shuffleBlock(text, 10)
>>> print shuffled_blocks
'edJigsawEnuriceLing,AndIInventMyNameIsMacryption'
>>> print block_order
[3, 1, 2, 0, 4]
>>>
Parameters:
  • block (string) - data block to be shuffled.
  • length (integer) - length of sub-block.
Returns:
(shuffled block, order of sub-blocks)

reversePartBlock2(self, block, slength, elength)

source code 

Function to reverse from M-th to N-th length of a string or a list where M < N. If M is zero or less than zero, reversal will start from the first element. If N is equal or more than the length of the block, the entire block from M will be reversed.

Parameters:
  • block (string) - data to be reversed.
  • slength (integer) - number of M-th items (from the front) to start reversing.
  • elength (integer) - number of N-th items (from the front) to start reversing.
Returns:
reversed block
