114      and backward hash functions to generate hash for the

117      dependent on the block size and number of start points. 
118       
119      Algorithm: 
120          1. forward hash <- hash(start) + [hash(start + 1)

122          2. backward hash <- hash(start) + [hash(start - 1)  
123          for start = startpoints to 0] 
124          3. Return forward hash + backward hash 
125       
126      @param filename: name of file to hash 
127      @type filename: string 
128      @param blocksize: block size for hash generation.  
129      Default = 1024 
130      @type blocksize: integer 
131      @param startpoints: defines the number of start point on 
132      the file for hash generation. Default = 10. 
133      @type startpoints: integer 
134      @param algorithm: algorithm to generate hash. Allowable 
135      options are {md5|sha1|sha244|sha256|sha384|sha512} 
136      @type algorithm: strong 
137      @return: generated hash in string 
138      ''' 
139      f = open(filename, 'rb') 
140      f.seek(-1, 2) 
141      fsize = f.tell() 
142      file_block_size = int(fsize/int(startpoints)) 
143      startpoints = [x*file_block_size
