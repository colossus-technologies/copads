set_instruction_version(ragaraja_version, instruction_set='ragaraja_instructions.txt')

source code 

Set active / usable set of Ragaraja instructions / operations for the current simulation.

Parameters:
  • ragaraja_version - defines the version of Ragaraja instructions to be used. Allowed values are {0 | 0.1 | 1}
  • instruction_set (string) - file name for user-defined instruction set usage. This will only be used when ragaraja_version=0. Default = ragaraja_instructions.txt. Please see ragaraja_instructions.txt for format.
Returns:
dictionary of ragaraja instructions as keys and instruction execution functions as values.

Since: version 0.4.1
