 1  """ 
 2  COPADS (Collection of Python Algorithms and Data Structures). 
 3   
 4  The main aim of COPADS is to develop a compilation of of Python data 
 5  structures and its algorithms, making it almost a purely developmental 
 6  project. Personally, I look at this as a re-usable collection of tools 
 7  that I can use in other projects. Therefore, this project is essentially 
 8  "needs-driven", except a core subset of data structures and algorithms. 
 9   
10  This project originated from 3 threads of thought. Firstly, while browsing 
11  through Mehta and Sahni's Handbook of Data Structures and Applications, I 
12  thought there might be utility to have a number of the listed data 
13  structures implemented in Python. Given my interest in biological data 
14  management, having a good set of data structures is always handy. The 2nd 
15  thread of thought came from Numerical Recipes. Again, I thought these 
16  algorithms will be handy to have and had started to translate some of 
17  them into Python during some overly energetic days. Finally, Python 
18  Cookbook had undergone 2 editions by 2008 and ActiveState had provided an 
19  online platform for Python Recipes which I found to be useful and can see 
20  how some of these recipes can be merged. Thus, COPADS is borned. 
21   
22  Project website: U{https://github.com/copads/copads} 
23   
24  License: Unless specified otherwise, all parts of this package, except 
25  those adapted, are covered under Python Software Foundation License 
26  version 2. 
27  """
