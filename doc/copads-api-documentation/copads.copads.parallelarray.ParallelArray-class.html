addDataList(self, fieldnames, values)

source code 

Method to add values using an ordered list of field names and the corresponding ordered list of values.

The lists of field names and and values need not be the complete set of field names in the array - all missing data (field names and values not provided) will be deemed as None, in order to maintain the same number of values in each field.

Values can be added to a non-existing field. In this case, a new field will be added and the values will be front-padded by None.

Parameters:
  • fieldnames (list) - ordered names of fields for values to be added to.
  • values (list) - ordered values to be added.
